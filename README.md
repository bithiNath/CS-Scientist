# CS-Scientist

## Build from Sources

1. Clone the repo

```bash
https://github.com/bithiNath/CS-Scientist.git
```

2. Intialize and activate virtual environment

```bash
virtualenv venv
```

3. Install dependencies

```bash
pip install -r requirements.txt
```
